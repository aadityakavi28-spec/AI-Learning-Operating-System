## Overview

AI Learning Operating System is an AI-powered platform that helps students learn from the internet while tracking their learning behavior and optimizing their study process.

## Features

- **Learning Search Engine** - Fetches resources like YouTube, articles, tutorials
- **Learning Environment** - Video viewer, article reader, AI assistant
- **Notes and Flashcards System** - Create and manage study materials
- **Learning Behavior Tracking System** - Track study patterns
- **AI Analysis Engine** - Analyze learning data
- **AI Learning Doctor** - Diagnosis system for learning issues
- **Personalized Study Plan Generator** - AI-driven study plans
- **Progress Analytics Dashboard** - Visualize learning progress
